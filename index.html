尝试使用Octopress来建站

Jan 28th, 2013 | Comments

最近发现我们更应该关注blog的内容本身，而不是它的展现形式。markdown是最近广泛使用于github以及stackoverflow等的程序员社区的一种标记语言，它能更好的让人们专注于写作内容的本身，而并非其展现形式。Blogging like hackers是octopress的宗旨，当下有很多人选择使用Octopress配合Github Pages建站。

众多新技术的风靡令我心里发痒，这是我重新搭建blog的主要动力。我希望能记录下来自己在学习中收获的点滴。

放假回家之后，主要折腾了下面一点事情：

  1. 因为下学期会有比较独立的web开发任务，我尝试了早有耳闻的django框架。不得不说的是其ORM, Admin Site等功能让人眼前一亮，而且还有很多可以复用的Apps，确实可以减少相当大的重复工作。缺点就是django是一个大包大揽的框架，对于大型应用来说可制定性估计不能满足要求。但是对于个人开发小网站来说绝对是非常好的选择。
  2. 学期末的时候决定把 Effective Java 重新认真读一下，不过好像读了几章之后又放下了。最近准备读一读然后写点笔记扔上来吧。
